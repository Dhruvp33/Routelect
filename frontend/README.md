# EV Route Planner - Frontend

Progressive Web App (PWA) built with React + Vite for intelligent EV route planning in India.

## 🛠️ Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **React Router** - Client-side routing
- **Tailwind CSS** - Utility-first styling
- **Leaflet** - Interactive maps (OpenStreetMap)
- **Zustand** - Lightweight state management
- **Vite PWA Plugin** - Progressive Web App features

## 📋 Prerequisites

- Node.js 18+ and npm/yarn
- Backend server running on `http://localhost:8000`

## 🚀 Installation & Setup

### 1. Install Dependencies

```bash
npm install
# or
yarn install
```

### 2. Start Development Server

```bash
npm run dev
# or
yarn dev
```

The app will be available at: `http://localhost:5173`

### 3. Build for Production

```bash
npm run build
# or
yarn build
```

This creates an optimized production build in the `dist/` folder.

### 4. Preview Production Build

```bash
npm run preview
# or
yarn preview
```

## 📱 PWA Features

The app is a fully functional Progressive Web App:

### Installation
- Click "Add to Home Screen" on mobile
- Install via browser prompt on desktop

### Offline Capabilities
- Map tiles cached for offline viewing
- Previously selected car data persists
- API responses cached (24hr)

### Service Worker
Automatically generated by Vite PWA plugin:
- Caches OpenStreetMap tiles (30 days)
- Caches API responses (24 hours)
- Network-first strategy for fresh data

## 🗺️ Map Integration

Using **Leaflet** with **OpenStreetMap** tiles:

```jsx
<MapContainer center={[23.0225, 72.5714]} zoom={8}>
  <TileLayer
    url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png"
    attribution='&copy; OpenStreetMap'
  />
</MapContainer>
```

### Map Features
- Click to set start/end locations
- Current location detection
- Marker popups for charging stations
- Route visualization with polylines

## 💾 State Management

Using **Zustand** for global state:

```js
// Store structure
{
  backendStatus: boolean,
  selectedCar: object | null,
  user: object | null,
  currentRoute: object | null,
  currentBatteryPercent: number
}
```

### Persistence
Selected car and user data persist to localStorage via Zustand middleware.

## 🔧 Environment Variables

No environment variables needed for frontend! 

Backend URL is hardcoded as `http://localhost:8000` for development.

For production, update API URLs in:
- `src/App.jsx`
- `src/pages/BrandSelection.jsx`
- `src/pages/ModelSelection.jsx`
- `src/pages/RoutePlanner.jsx`

Or create `src/config.js`:
```js
export const API_BASE_URL = import.meta.env.VITE_API_URL || 'http://localhost:8000'
```

## 🚨 Common Issues & Solutions

### Map Not Loading
```bash
# Clear browser cache and reload
# Check Leaflet CSS is imported in index.html
```

### Backend Connection Failed
- Ensure backend is running on port 8000
- Check CORS settings in backend
- Open browser console for errors

### PWA Not Installing
- Must be served over HTTPS (or localhost)
- Check manifest.json is accessible
- Clear service worker cache

### Build Errors
```bash
# Clear node_modules and reinstall
rm -rf node_modules package-lock.json
npm install
```
